# Proyecto_IoT_Luis_Alejandro
El objetivo principal es supervisar las condiciones del entorno que afectan el desarrollo vegetal (como temperatura, humedad y luz) para optimizar el cuidado de las plantas. La arquitectura se basa en un modelo Publicador/Suscriptor, donde varios nodos sensores envían información a un bróker, y un nodo recolector almacena y visualiza los datos.
